This course is the 1st step of your first step towards becoming web developer

Every website has 2 parts

Front-end

It is the part that you see in your browser and interact with all the visual aspects.

Back-end

It is the part that powers the front end. It's mainly about storing data in databases and providing it to the fornt page.

Fontend developer uses HTML, CSS, Javascript.



HTML = Hypertext Mark up language. It is used to make the building block of our webpages.

CSS = Cascading Style Sheet. It is used to styling our webpages and make our pages beautiful

Javascript is used to adding functionality to web pages.

Building on real world is like webpage on internet, it has skeleton or sturcture(HTML).

It can have preety walls, windows , tiles (CSS)

It can have certain functionality. For example when we press the elevator button, it opens.(Javascript)


Suppose, we want to build a website like tweeter. For each tweet we would like to make a layout like the follwing image.


First we have to make building blocks of this layout. Img, text, icon these are the building blocks.


We use html to add this building blocks to our webpage.

After HTML we use CSS to give it visual effects.

We can make the text bold, make the image round , give some padding.

CSS is all about asthetics and visual effcets.By using css we can also make beautiful animations.

By using Javascript we can add functionality of behaviour of our page. For example we can like a tweet

Javascript is a programming language while HTML , CSS is not. We cannot use them to tell computer what to do. They are just used to make and style the webpage.


Every webpage on internet are build using these three languages. So better you learn and understand these languages and their features the better you will be at front-end development.
