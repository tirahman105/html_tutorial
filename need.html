After installing VS code we have to download some extensions. These will help us and make our task more easy.

Prettier - Code formatter

This will help to formate our code. To download this click on the extension tab and type Prettier in search bar. Then install it.

Live server

This will help us to see live preview. To download this click on the extension tab and type Live server in search bar. Then install it.
